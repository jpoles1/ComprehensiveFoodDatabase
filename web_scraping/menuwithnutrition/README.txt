About:
This program scrapes all of the foods into one csv file (in this directory it is `menu_with_nutrition_scrape.csv`).
There is also a folder `csvs` that has all of the foods split into 1000 entry files.
There are a total of 203086 entries.

Dependencies:
- re
- pandas
- bs4
- urllib

How to run:
In a *nix terminal, run
`python3 scrape.py`
